# Advanced-Animals-Care
**An mods for Project Zomboid game.**

<ins>Enhances the ranch UI</ins>

### Hightlights status:

- Hunger.
- Thrist.
- Stress.
- Health.

### Status display in panel:

- Health.
- Pregnancy.
- Pettable.

**You can change the color of the highlights in the mod options.**
